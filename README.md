1. **Clone the repository**

   ```bash
   git clone https://github.com/joao-tambue/newsaggregator.git
   cd newsaggregator
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```
